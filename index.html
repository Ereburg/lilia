Команды Git

  1. git init - инициализация пустого репозитория
  2. git add . - добавить файлы для отслеживания
  3. git commit -a -m "первый коммит" - для именования сохранений
  4. git status - для проверки текущего состояния файлов
  5. git branch - для создания новой ветки
  6. git checkout - для переключения на ветку
  7. git checkout -b - для создания ветки и переключения на нее
  8. git merge - для слияния веток
  9. git reset HEAD~n (n - число коммитов для удаления) - для удаления коммитов
  10. clear - очистить консоль
